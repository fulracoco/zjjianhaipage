# ShipServ 供应商注册准备指南

> 适用企业：镇江舰海船舶配件有限公司  
> 核对日期：2026-07-16

## 1. 注册结论

ShipServ 目前允许供应商免费注册 `Trade Basic`。基础注册没有公开要求企业必须先持有 ISO、船级社认证或 IMPA 会员资格。

官方注册链接：

- [ShipServ Supplier Sign Up](https://www.shipserv.com/sign-up)

镇江舰海船舶配件有限公司现有的 ISO 9001、出口资质、产品资料、工厂资料和多语言网站，已经能够支持基础供应商档案申请。若需要展示证书、直接联系方式、更多品牌与分类并提升搜索排名，应进一步评估付费 `Premium Profile`。

## 2. 基础注册信息

ShipServ 当前注册表单首先要求以下信息：

- 公司英文名称
- 公司类型
- 公司英文介绍
- 主要行业，选择 `Marine`
- 企业联系人和登录邮箱
- 国家、地址和电话等企业资料
- 产品分类和供应品牌
- 了解 ShipServ 的渠道
- 同意平台条款及隐私政策

本公司类型建议选择：

```text
Manufacturer - Equipments
```

除非公司确实以代理或转售业务为主，否则不应选择经销商类型。

## 3. 资质与证书

| 资料或证书 | 基础注册要求 | 准备建议 |
|---|---|---|
| 营业执照 | 通常用于企业真实性核验 | 必备 |
| ISO 9001 | 不是基础注册强制项 | 强烈建议提供 |
| 海关或出口备案 | 不一定在注册时强制 | 出口企业应准备 |
| 产品检测报告 | 不强制 | 强烈建议提供 |
| 材料证明 | 不强制 | 重要订单建议提供 |
| CCS、DNV、LR、ABS、BV 等船级社认证 | 不是平台基础注册条件 | 买家或具体项目可能要求 |
| 品牌授权书 | 仅在声称授权代理时需要 | 不得虚假声明 |
| ISO 14001、ISO 45001 | 不强制 | 可提升 ESG 和供应商形象 |
| IMPA 会员或 IMPA 编码 | 不强制 | 产品分类时可参考 |

## 4. Basic 与 Premium 的区别

根据 ShipServ 官方说明，`Trade Basic` 可用于：

- 免费建立基础供应商档案
- 接收 RFQ 和订单
- 在线报价和确认订单
- 接收交易通知

基础档案存在以下限制：

- 可展示的品牌和产品分类数量有限
- 不能显示直接联系方式
- 不能上传文件或证书
- 不能使用平台广告功能

`Premium Profile` 通常增加以下能力：

- 上传 ISO 9001、产品目录和其他文件
- 展示公司 Logo 和详细介绍
- 展示更多或不限量的品牌与分类
- 提高 ShipServ 搜索结果中的排名
- 使用 Spotlight Listing 等推广功能
- 展示直接联系方式及更多可信度信息

Premium 属于付费服务，ShipServ 当前官网倾向于让供应商联系销售或申请演示，没有公开统一固定价格。购买前应要求销售提供正式报价、服务周期、品牌与分类额度、退出条款和预期曝光数据。

## 6. 品牌与机型填写规范

本公司产品涉及 MAN B&W、Wärtsilä、Daihatsu、Yanmar、Mitsubishi 等机型。若没有品牌方正式授权，不应将公司描述为这些品牌的授权代理商、授权经销商或原厂供应商。

建议使用以下英文表达：

```text
Manufacturer of compatible replacement parts
Compatible with selected MAN B&W engine models
Suitable for selected Wärtsilä engine models
Drawing-based manufacturing available
```

不应使用以下表达，除非能够提供有效授权文件：

```text
Authorized Distributor
Official Dealer
OEM Supplier for MAN B&W
Genuine Wärtsilä Parts
```

只有能够提供品牌授权书或经销协议时，才应申请 ShipServ 的 verified distributor badge。

## 7. 建议选择的产品方向

注册时优先选择与实际业务一致的船舶和发动机零部件分类，例如：

- Marine Diesel Engine Parts
- Engine Components
- Intake and Exhaust Valves
- Valve Seats
- Valve Spindles
- Replacement Engine Parts
- OEM and Drawing-based Manufacturing

具体名称应以 ShipServ 注册后台提供的正式分类为准。不要为了增加曝光选择与实际产品无关的分类。

## 9. 产品资料填写参考

每个产品条目建议包含：

1. 产品名称，例如 `Marine Engine Exhaust Valve`。
2. 适配品牌和机型。
3. 零件号或图纸编号。
4. 材料和强化工艺。
5. 尺寸范围。
6. 产品用途和工作位置。
7. OEM 或来图加工能力。
8. 询价时买家需要提供的资料。
9. 包装和发货方式。
10. 可提供的质量文件。

询价说明参考：

```text
Please provide the engine brand, engine model, part number, drawing or key
dimensions, material requirements and order quantity so that we can verify
the specification.
```

## 10. 支付与合规功能的额外要求

如果后续使用 Marcura Pay、企业付款或其他合规服务，可能需要单独完成 KYC 或 AML 审核。常见资料包括：

- 营业执照
- 企业银行账户
- 法定代表人信息
- 最终受益人信息
- 注册地址和经营地址
- 税务资料
- 主要贸易国家和收款币种
- 制裁、船舶和最终用户合规信息

这些要求属于支付或合规服务，不等同于基础供应商档案注册条件。
